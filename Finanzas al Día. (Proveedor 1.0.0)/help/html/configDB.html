Ventana "Configuración de la Base de Datos"

En esta ventana se configuran los datos necesarios para acceder a la Base de Datos de donde se importará la información del sistema, así como se configura la ubicación de los documentos. A continuación se describen cada uno de los parámetros a introducir:

Servidor: "Dirección IP donde está ejecutándose la Base de Datos"

Nombre: "Nombre de la Base de Datos"

Puerto: "Puerto que utiliza el servidor MySQL "

Usuario: "Usuario que utiliza el servidor MySQL"

Contraseña: "Contraseña de la Base de Datos"

Acceso a Documentos: "Ruta de los documentos"

Luego de llenar los datos necesarios se debe hacer clic en el botón Importar datos para guardar la configuración especificada e importar de la Base de Datos los documentos que serán visualizados en la aplicación, de esta manera queda actualizada la aplicación proveedor. Se debe dar clic en el botón Salvar si se desea guardar la configuración de la ruta donde se encuentran los documentos.
